Encore des paquets Debian et Ubuntu

with 2 comments

Les paquets Debian pour TeX Live 2009, dont j’annonçais dernièrement l’arrivée dans les dépôts sid, sont désormais disponibles dans la version testing de la distribution. Par ailleurs, des paquets pour Ubuntu sont disponibles, soit pour Karmic (dépôt non officiel), soit pour Lucid dans les dépôts officiels. Ainsi, TeX Live 2009 sera intégré aux prochaines versions stables de ces deux distributions.Je rappelle les principales différences entre ces paquets et une TeX Live nature (mis à part la facilité d’installation) : la version Debian/Ubuntu ne contient pas tlmgr, le gestionnaire de paquets TeX Live permettant une mise à jour en continu de paquets ; en revanche, biblatex, le nouveau gestionnaire de bibliographies avec BibTeX (encore en version beta) est inclus, et LuaTeX, le moteur TeX du futur, bénéficie de mises à jour en continu.

TeXworks

Par ailleurs, le nouvel éditeur intégré offrant une synchronisation native entre le source et le PDF, TeXworks, qui est distribué avec TeX Live sous Windows et avec MacTeX sous OS X, est également disponible en paquet Debian pour testing et en paquet Ubuntu pour Lucid.

La version incluse dans Ubuntu est complète concernant les paquets (La)TeX, mais ne contient pas tlmgr : les mises à jour se font comme pour les autres paquets (par APT ou son interface Synaptic). Elles sont bien moins fréquentes que celles par tlmgr, mais avez-vous besoin des mises à jour quotidiennes ? Si oui, il vous faudra installer TeX Live vous-même, ce qui n’est pas très difficile dans l’absolu, mais n’est pas forcément le plus facile si vous débutez avec Linux.

    Concernant l’éditeur, aucun n’est fournit directement avec TeX Live sous Linux, mais plusieurs sont disponibles en paquets Ubuntu : TeXworks et TeXmaker, qui existent aussi sous les autres plate-formes, Kile qui est peut-être le plus populaire sous Linux, et bien d’autres.

    Enfin, il n’y a pas de possibilité d’installer à la volée les paquets manquants comme sous MikTeX. Je recommande d’installer TeX Live en entier pour ne pas avoir de question à se poser (paquet texlive-full dans Ubuntu).
